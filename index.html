AMPS: Automated Manufacturing of PCBs

AMPS aims to combine all the PCB manufacturing steps in a single automated machine.

Manufacturing PCB prototypes rapidly, control all the processes from a user friendly interface and keep the design data secure.

Single & Double sided PCBs are our current focus.(Ability to manufacture Multilayer PCBs coming later.)
Laser is used to expose circuit traces.
Drilling & Milling of PCB vias possible using an inbuilt drilling head.
Etching, Resist Stripping & Hole Plating is done by an automated chemical processing unit.
Automatic Optical Inspection is done using an in-built camera.

FAQ

What is a AMPS?
AMPS is a company attempting to automate and miniaturize PCB production setup. Prototype PCBs are currently manufactured with expensive manual prototyping machines or by sending designs to PCB manufacturing Factories. Electronics enthusiasts, researchers and small manufacturers rely on these factories to produce their PCBs since setting up their own PCB manufacturing setup is very expensive and prototyping PCBs in-house is Laborious and time consuming.
Why do I need an AMPS machine when I can use a milling machine or a Fiber Laser to make my own PCBs?
Making very thin traces and conductive vias is not possible using Milling and Fiber lasers. Also these methods require the user to make multiple prototypes using trial and error to get the settings right. Our machines are automated and produce Precise and Repeatable production grade PCBs.
